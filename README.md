# 🏋️‍♀️ Fitness Tracker AI

A modern full-stack AI-powered fitness tracking application that helps users manage their health by tracking calories, workouts, and nutrition. This project leverages Google Gemini AI to analyze food images and automatically estimate calorie intake.

---

## 🚀 Overview

Fitness Tracker AI is designed to simplify health tracking by combining traditional fitness logging with AI-powered food recognition. Users can set goals, track their daily activities, and gain insights into their calorie consumption effortlessly.

This project is ideal for developers looking to build a real-world AI-integrated full-stack application for their portfolio.

---

## ✨ Features

* ✅ Set and manage daily fitness goals
* ✅ Track food intake (calories consumed)
* ✅ Track fitness activities (calories burned)
* ✅ User authentication (Sign up / Sign in)
* ✅ Update user profile information
* ✅ Upload food images for calorie estimation
* ✅ AI-powered food recognition using Google Gemini AI
* ✅ Automatic calorie calculation from food images
* ✅ Fully deployed frontend and backend

---

## 🛠️ Tech Stack

### Frontend

* React JS
* Tailwind CSS

### Backend

* Strapi (Headless CMS & API)

### AI Integration

* Google Gemini AI (for food image analysis)

### Deployment

* Frontend: Vercel
* Backend: Strapi Cloud

---

## 📸 How It Works

1. User signs up or logs into the application
2. Sets daily fitness goals
3. Logs food intake manually or uploads food images
4. AI analyzes the uploaded image and estimates calories
5. Tracks workouts and calories burned
6. Displays overall progress and insights

## 🧠 AI Food Analysis

This application integrates Google Gemini AI to:

* Detect food items from uploaded images
* Estimate portion sizes
* Calculate approximate calorie values

This removes the need for manual calorie tracking and improves user experience.

## ⚙️ Installation & Setup

### Prerequisites

* Node.js
* npm or yarn
* Strapi CLI

### Clone the Repository

```bash
git clone https://github.com/your-username/fitness-tracker-ai.git
cd fitness-tracker-ai
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run develop
```
